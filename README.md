# Longest Valid Parentheses - Moglix Online Shortlisting Round

This repository contains my solution for the **Longest Valid Parentheses** programming problem as part of the **Moglix Online Shortlisting Round**.

## Problem Statement

Given a string containing only '(' and ')', return the length of the longest valid (well-formed) parentheses substring.

### Example

Input:
```
(() 
```

Output:
```
2
```

Input:
```
)()())
```

Output:
```
4
```

## Solution

- **Language:** Java
- **Approach:** Stack
- **Time Complexity:** O(n)
- **Space Complexity:** O(n)
